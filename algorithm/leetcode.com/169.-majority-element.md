# 169. Majority Element

## Make hash and sort

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        result = {n: 0 for n in set(nums)}
        for num in nums:
            result[num] += 1
        return sorted(result.items(), key=lambda x: -x[1])[0][0]
```

## Use Counter

```python
from collections import Counter

class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        count = Counter(nums)
        return count.most_common()[0][0]
```

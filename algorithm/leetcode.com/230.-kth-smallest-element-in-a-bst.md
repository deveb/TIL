# 230. Kth Smallest Element in a BST

```python
class Solution:
    def kthSmallest(self, root: TreeNode, k: int) -> int:
        stack = []
        # in-order, dfs
        while True:
            while root:
                stack.append(root)
                root = root.left
            # if not stack:
            #     break
            root = stack.pop()
            k -= 1
            if not k:
                return root.val
            root = root.right
```

# 239. Sliding Window Maximum

## Naive

```python
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        if not nums:
            return []
        return [max(nums[i:i + k]) for i in range(len(nums) - k + 1)]
```

O\(Nk\)

## Naive, Remove max\(\)

```python
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        if not nums:
            return []
        answer = []
        max_idx = None
        for i in range(len(nums) - k + 1):
            last = i + k 
            if max_idx and i <= max_idx <= last - 1:
                if nums[max_idx] < nums[last - 1]:
                    max_idx = last - 1
            else:
                max_idx = i
                for i in range(i + 1, last):
                    if nums[max_idx] < nums[i]:
                        max_idx = i
            answer.append(nums[max_idx])
        return answer
```

O\(Nk\)

## Dynamic programming

![](../../.gitbook/assets/image%20%2814%29.png)

```python
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        if not nums:
            return []

        # left = max element from block start to j
        left = [None] * len(nums)
        for i in range(len(nums)):
            if i % k == 0:
                left[i] = nums[i]
            else:
                left[i] = max(left[i-1], nums[i])

        # right = max element from block end to j
        right = [None] * len(nums)
        right[-1] = nums[-1]
        for i in range(len(nums)-2, -1, -1):
            if i % k == k - 1:
                right[i] = nums[i]
            else:
                right[i] = max(right[i+1], nums[i])

        return [max(left[i + k -1], right[i]) for i in range(len(nums) - k + 1)]
```

O\(N\)

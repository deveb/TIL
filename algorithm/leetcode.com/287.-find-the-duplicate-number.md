# 287. Find the Duplicate Number

## **Sort**

```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        nums.sort()
        prev = None
        for n in nums:
            if prev is not None and n == prev:
                return n
            prev = n
```

O\(nlogn\)

## Using Counter

```python
from collections import Counter

class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        return Counter(nums).most_common()[0][0]
```

O\(n\)

```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        tortoise = nums[0]
        hare = nums[0]
        # phase 1
        while True:
            tortoise = nums[tortoise]
            hare = nums[nums[hare]]
            if tortoise == hare:
                break
        # phase 2
        ptr1 = nums[0]
        ptr2 = tortoise
        while ptr1 != ptr2:
            ptr1 = nums[ptr1]
            ptr2 = nums[ptr2]
        return ptr1
```

O\(n\)

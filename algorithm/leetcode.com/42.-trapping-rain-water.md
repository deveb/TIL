## Brute Force

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        water = 0 
        for i in range(1, len(height) -1):
            left_top = max(height[:i])
            right_top = max(height[i + 1:])
            
            maxwater = min(left_top, right_top)
            if maxwater > height[i]:
                water += maxwater - height[i]
                
        return water
```

O\(n^2\)
